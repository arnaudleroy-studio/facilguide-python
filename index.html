facilguide

A Python package for accessing Facil.guide's catalog of simplified tech guides for seniors.

Version 0.1.0
Facil.guide PyPI GitHub Install API Reference

What is this?

Facil.guide provides clear, step-by-step tech guides designed for seniors and anyone who prefers simple instructions. Topics cover smartphones, tablets, internet safety, social media, and more.

The facilguide Python package gives you access to the full guide catalog directly in your code. It includes 15 guides across 8 categories, available in 5 languages. No API key needed, no internet connection required after install.

Installation

Install from PyPI with pip. Requires Python 3.9 or newer. No dependencies.

pip install facilguide

Quick Start

After installing, import facil and start exploring the guide catalog.

# Import the package
import facil

# See all available guides
all_guides = facil.guides()
print(f"{len(all_guides)} guides available")
# 15 guides available

# Get guides in Spanish
spanish = facil.guides(lang="es")

# Get security-related guides
security = facil.guides(category="security")

# Search by title
results = facil.search("whatsapp")
print(results[0]["title"])
# How to Set Up WhatsApp

# List all languages
for lang in facil.languages():
    print(f"{lang['name']}: {lang['url']}")

# List all categories
for cat in facil.categories():
    print(f"{cat['name']}: {cat['description']}")

API Reference

The package provides four functions. All data is bundled locally, so these work offline.

facil.guides(*, lang=None, category=None)

Returns a list of guide dictionaries. You can filter by language, category, or both.

lang (str, optional): Language code. One of en, es, fr, pt, it.
category (str, optional): Category slug. See the categories table below.
Returns: List of dicts, each with slug, title, category, difficulty, and languages.

facil.categories()

Returns a list of all guide categories.

Returns: List of dicts, each with slug, name, and description.

facil.languages()

Returns all supported languages with their Facil.guide URLs.

Returns: List of dicts, each with code, name, and url.

facil.search(query, *, lang=None)

Search guides by title. Case-insensitive.

query (str): The search term.
lang (str, optional): Limit results to a specific language.
Returns: List of matching guide dicts.

More Examples

Combine filters

# Beginner social media guides in French
french_social = facil.guides(lang="fr", category="social")
for g in french_social:
    print(g["title"])

Count guides per language

for lang in facil.languages():
    count = len(facil.guides(lang=lang["code"]))
    print(f"{lang['name']}: {count} guides")

# English: 15 guides
# Spanish: 14 guides
# French: 13 guides
# Portuguese: 11 guides
# Italian: 10 guides

Build a simple guide menu

for cat in facil.categories():
    guides_in_cat = facil.guides(category=cat["slug"])
    print(f"\n{cat['name']} ({len(guides_in_cat)} guides)")
    for g in guides_in_cat:
        print(f"  - {g['title']}")
